# ResumeCraft – AI-powered Resume Optimizer

## Overview
ResumeCraft is an intelligent resume optimization platform that uses AI and Natural Language Processing (NLP) to help users create professionally formatted resumes that pass ATS (Applicant Tracking Systems) checks. It offers live LaTeX preview, customizable templates, and multi-format export (PDF, DOCX).

## Features
- AI-based content enhancement & keyword optimization
- Live LaTeX preview with instant formatting updates
- Export resumes in PDF and DOCX formats
- User-friendly web interface for seamless resume creation
- Automated parsing & formatting for professional results
- Cloud deployment for anytime, anywhere access
- CI/CD automation ensuring continuous improvements and stability

## Tech Stack
- **Frontend:** Next.js, React
- **Backend:** GraphQL API
- **AI & NLP:** HuggingFace LLM API
- **Database:** PostgreSQL
- **Containerization & Deployment:** Docker, AWS

## Getting Started

### Clone the repository
```
git clone https://github.com/gaurav0330/ResumeCraft-AI.git
cd ResumeCraft-AI
npm install
```

### Start the development server
```
npm run dev
```

## Usage
Open http://localhost:3000 in your browser and start creating or optimizing your resume with AI-powered suggestions and live preview.
